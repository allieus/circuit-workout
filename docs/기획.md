# 타이머 + 운동 가이드 기획

핵심 설계 원칙: **운동 중에는 폰을 쳐다보지 않아도 되게 한다.**
새 동작을 기억하지 못하는 사용자를 위해 앱이 눈(화면)과 귀(음성)로 대신 알려주는 구조.

## 1. 타이머 상태 흐름 (5단계)

```
준비(10초) → 운동(40초) → 휴식(20초) → ... → 라운드 휴식(60초) → 완료
```

| 단계 | 역할 |
|---|---|
| 준비 | 시작 직후 카운트다운. 첫 동작 이름 + 폼 포인트 미리 표시, 자세 잡을 시간 |
| 운동 | 현재 동작 수행. 화면 전체가 패턴 색으로 변경 |
| 휴식 | 쉬는 시간이 아니라 "다음 동작 예습 시간"으로 설계 |
| 라운드 휴식 | 5동작 한 바퀴 후 긴 휴식. 다음 라운드 전체 순서 리스트 복습 |
| 완료 | 총 소요 시간·수행 요약 |

## 2. 타이밍별 가이드 콘텐츠

| 구간 | 화면(눈) | 음성(귀) |
|---|---|---|
| 준비 | 첫 동작 이름 + 폼 포인트 | "준비하세요. 첫 동작, ○○. (폼 포인트)" |
| 운동 시작 | 동작 이름 크게 + 폼 포인트 | "시작!" |
| 운동 절반 | 진행 바 | "절반이에요" |
| 운동 끝 3초 전 | 숫자 강조 | 비프음 3회 |
| 휴식 | "다음: ○○" + 폼 포인트 | "다음은 ○○입니다. (폼 포인트)" |
| 라운드 휴식 | 다음 라운드 5개 동작 순서 | "n라운드 완료. 잠시 쉬세요" |
| 완료 | 요약 | "모든 라운드 완료! 수고했어요" |

음성: Web Speech API 한국어 TTS. 브라우저 내장, 별도 비용 없음. 토글 제공.

## 3. 컨트롤

- 일시정지 / 재개
- 건너뛰기: 즉시 다음 단계로 (secondsLeft=0 → advance 경로 공유)
- 동작 교체(↻): 타이머 도중에도 같은 패턴의 다른 동작으로 즉석 교체
  - 운동 중 → 현재 동작 교체 / 휴식 중 → 다음 동작 교체
  - 기구가 없거나 컨디션이 안 좋은 날 대응

## 4. 설정

라운드(1–8) / 운동(10–120초) / 휴식(5–90초) / 준비(5–15초) / 라운드 휴식(30–120초) / 음성 on·off
