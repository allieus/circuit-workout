# 케틀벨 핵심 7가지 (참고 자료)

출처: 유튜브 "하루 종일 지치지 않는 체력을 위한 운동 7가지 (케틀벨 종합편)"
(https://youtu.be/GUBlDoxLAFs) — 자막 기반 정리. defaults.js 케틀벨 동작의 근거 문서.

## 7가지 동작과 습득 순서

앞 단계 무게가 가벼워지면 다음 단계로 진행하는 구조.

1. **양손 케틀벨 스윙** — 모든 응용의 기초. 발과 케틀벨이 정삼각형 → 힙 힌지 →
   가랑이 사이로 던지기(손목이 사타구니에 닿게) → 일어서며 발사, 어깨 높이까지.
   골반 높이로 떨어질 때 다시 힌지. 입문 무게: 여성 ~9kg, 남성 ~13.6kg
2. **한팔 케틀벨 스윙** — 코어 강화↑, 같은 무게도 더 무겁게 작용.
   빈손은 케틀벨 손잡이를 터치 — 몸통 비틀림(허리 부상) 방지 장치
3. **케틀벨 클린** — 연습 단계: 명치 높이 스윙 → 미드풀(팔꿈치 ㄴ자, 엄지 위) →
   5회째에 턱에 어퍼컷 찌르듯. 손목 꺾임 주의, 아프면 한 달 정도 아대
4. **케틀벨 프레스** — 클린 상태에서 수직으로. 전완 수직 유지 못 하면 무게 낮추기
5. **케틀벨 스쿼트** — 클린 랙 상태로 앉았다 일어나기. 편측이라 코어 강화 큼
6. **케틀벨 스내치** — 어깨 강화·칼로리 소모가 스윙보다 큼. 한팔 스윙보다 4~8kg 가볍게.
   연습: 이마 높이 스윙 → 하이풀 → 손 위로 뻗기. 내릴 땐 처음엔 프레스로, 익숙해지면 드랍
7. **터키시 겟업** — 전신, 특히 어깨·코어. 케틀벨에서 시선 떼지 않기.
   맨손 연습 충분히 → 4~4.5kg부터

## 프로그램 (앱 프리셋 후보)

- **10초/50초 법칙**: 스윙·스내치용. 1분을 1세트로, 10초 운동(≈5회) + 50초 휴식.
  최소 10세트. 반복수 대신 세트수를 늘리고, 30세트가 쉬워지면 무게 증가
- **무빙 타겟 컴플렉스**: 클린 2 → 프레스 1 → 스쿼트 1, 손 바꿔 반복. 총 9세트, 약 30분.
  프레스 6~8회 가능한 무게
- **딥 식스(Deep 6)**: 한팔 스윙 → 스내치 → 클린 → 프레스 → 스쿼트 → 겟업 1회씩 연결.
  고강도 — 7동작 충분히 연습 후에만. 겟업은 항상 1회 고정
- **주 5일 스케줄**: 월(유산소 약 — 스윙 or 스내치 + 겟업, 중량 ÷2) /
  화(근력 강 — 무빙 타겟, 본 중량) / 수(휴식 or 딥식스 가볍게) /
  목(유산소 강 — 본 중량) / 금(근력 약 — 무빙 타겟, 중량 ÷2)
